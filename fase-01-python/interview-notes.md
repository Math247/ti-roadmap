# O que é uma variável?
É um espaço em memória para armazenar dados

# Em Python, variáveis armazenam valores ou referências?
Referências

# Qual a diferença entre str, int, float e bool?
str é string, um texto. int são numeros inteiros, float são numeros com ponto flutuante. bool armazena True ou False.

# Por que input() sempre retorna str?
Por que recebe o que vem digitado pelo usuário, que é sempre texto.

# O que é conversão de tipo?
Altera o tipo da variavel, ex: de string para int, e etc.

# Qual a diferença entre = e ==?
= é atribuição. == é comparação.

# O que é precedência de operadores?
É a ordem que o Python usa para decidir qual operação resolver primeiro.

# Qual a diferença entre import math e from math import sqrt?
import math importa tudo do modulo math. from math vai importar apenas uma função.

# O que significa dizer que uma string é indexada?
Cada caractere dentro da string tem uma posição numérica.

# O que frase[0:5] retorna?
retorna tudo na string que começa da posição 0, e termina na posição 4

# Qual a diferença entre strip(), upper(), lower() e replace()?
strip -> retira os espaços do inicio e do final
upper -> deixa tudo em maiusculo
lower -> deixa tudo em minusculo
replace -> substitui algo na string por outra coisa que você digitar
