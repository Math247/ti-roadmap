# O que é uma variável?
É um espaço em memória para armazenar dados

# Em Python, variáveis armazenam valores ou referências?
Referências

# Qual a diferença entre str, int, float e bool?
str é string, um texto. int são numeros inteiros, float são numeros com ponto flutuante. bool armazena True ou False.

# Por que input() sempre retorna str?
Por que recebe o que vem digitado pelo usuário, que é sempre texto.

# O que é conversão de tipo?
Altera o tipo da variavel, ex: de string para int, e etc.

# Qual a diferença entre = e ==?
= é atribuição. == é comparação.

# O que é precedência de operadores?
É a ordem que o Python usa para decidir qual operação resolver primeiro.

# Qual a diferença entre import math e from math import sqrt?
import math importa tudo do modulo math. from math vai importar apenas uma função.

# O que significa dizer que uma string é indexada?
Cada caractere dentro da string tem uma posição numérica.

# O que frase[0:5] retorna?
retorna tudo na string que começa da posição 0, e termina na posição 4

# Qual a diferença entre strip(), upper(), lower() e replace()?
strip -> retira os espaços do inicio e do final
upper -> deixa tudo em maiusculo
lower -> deixa tudo em minusculo
replace -> substitui algo na string por outra coisa que você digitar

# O que find() retorna quando não encontra o texto?
-1

# O que é uma condição em programação?
São diferentes caminhos que o programa pode ou não seguir. São definidos por if ou else, e só um deles será seguido.

# Qual a diferença entre if, elif e else?
if é uma condição. elif é acionado quando a condição do if é falsa, e adiciona outra condição. else é acionado quando a condição, seja do if ou do elif ou dos dois, são falsas

# Para que serve a indentação em Python?
são blocos de codigos que dependendo da sua posição, vao indicar se estao dentro de uma condição ou nao. se estiver dentro de um if, estara mais para a direita, se nao vai ficar totalmente colado a esquerda.

# Qual a diferença entre and e or?
and é um operador lógico que exige que todas as condições sejam verdadeiras. or é é um operador lógico que exige que apenas uma a condição seja verdadeiras.

# O que o operador not faz?
operador de negação. se uma condição seria verdadeira, e entraria num if, se ela tem o not antes, sera falsa e ira para o else.

# O que acontece se uma condição do if for True?
o codigo identado no if sera executado

# O que acontece se nenhuma condição de um if/elif for True?
nada acontece. se tiver um else, a condição identada no else sera executada.
