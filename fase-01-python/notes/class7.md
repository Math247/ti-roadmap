# Operadores Aritméticos

'+'= adição
'-' = subtração
'*' = multiplicação
'/' = divisão
'**' = potencia
'//' = divisão inteira
'%' = resto da divisão

'=' -> atribuição, recebe

'==' -> para operações aritméticas

## Ordem de precedencia dos operadores
Primeiro ao ultimo
1 - ()
2 - **
3 - *, /, //, % -> quem aparecer primeiro
4 - +, -

## Com strings

o + concatena

'Oi' + ' tudo bem' = 'Oi tudo bem'
