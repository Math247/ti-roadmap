# Condições (Parte 1)

Identação: Um espaço no bloco para dizer que certa parte do codigo esta dentro da condição.

if -> 'se'
else -> 'senão'

So o if -> estrutura condicional simples

if e else -> estrutura condicional composta

Numa condição, ou o bloco verdadeiro é executado, ou o bloco falso. Nunca os dois juntos. 

Todo comando q estiver colado do lado esquerdo, sempre sera executado. Se estiver identado, depende da condição.

## Condição simplificada:

print('carro novo' if tempo <= 3 else 'carro velho')
