# Módulos

para importar um módulo, utilizamos import. 

import modulo

se eu quiser, posso importar apenas algo especifico de um import:

from modulo import ....

o de cima eh generalista, o debaixo eh mais especifico, o q economiza memória.
exemplo:

'import math' -> traz algumas funcionalidades matematicas extras.

quando usamos o import math, precisamo usar math.sqrt() para utilizar as funções. mas se usarmos o from math import sqrt, so precisamos colocar sqrt(), pois ja importamos direto essa função.
